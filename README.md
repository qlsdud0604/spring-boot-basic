# 스프링 부트 프로젝트

## 기술 스택
* IDE
```
 - IntelliJ IDEA
```
* Language
```
 - Java
 - HTML
```
* Back-end
```
 - Spring Boot
 - Spring Data JPA
```
* Database
```
 - H2
```
</br>

## 프로젝트 목적
* 웹 애플리케이션을 개발할 때 어떤 기술들이 어떤 식으로 사용되는지 파악한다.
* 스프링 웹 애플리케이션을 만드는 핵심 사이클을 경험해 봄으로써 스프링 개발의 전반적인 감을 잡는다.
</br>

## 프로젝트 기능
* 사용자 이름을 통한 회원 가입 기능
* 회원 목록 조회 기능
</br>
